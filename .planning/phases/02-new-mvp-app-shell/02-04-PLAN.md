       - Set `style={{ background: '#11131a' }}` on the Canvas
       - Inside Canvas: `<Suspense fallback={null}>`, then `<Planet />`, `<PostStars posts={posts} />`, `<GalaxyBackground />`, `<Stars />` (from drei — ambient background stars)
       - Props: `{ posts?: AnalyzedPost[] }`
       - Must respect `prefers-reduced-motion`: if reduced motion, skip planet rotation animation

    2. Create `components/galaxy/Planet.tsx` — central spinning planet:
       - Use `useFrame` from R3F to rotate the mesh: `ref.current.rotation.y += 0.0005` (120s per revolution = ~0.00087 rad/frame at 60fps, use 0.0005 for subtle)
       - Sphere geometry: radius 1.5, widthSegments 64, heightSegments 64
       - Material: `MeshStandardMaterial` with color `#1a1d26` (surface tier 2), emissive `#374956` (secondary) at low intensity (0.1)
       - Add ambient light (intensity 0.3) and a point light (position [5, 5, 5], intensity 0.8, color `#E8D5A3` cream-gold)
       - Per UI-SPEC: minimum 320px visual diameter. The sphere radius + camera distance achieves this.
       - Use `useReducedMotion` from `motion/react` — if true, do not rotate

    3. Create `components/galaxy/PostStars.tsx` — posts as stars visualization (per D-08):
       - Props: `{ posts?: AnalyzedPost[] }`
       - If no posts or empty array, render nothing (empty state handled at page level)
       - For each post, render a small sphere or point:
         * Position: distribute around the planet in a ring/cloud pattern. Use polar coordinates with some randomization.
         * Brightness (opacity) based on ER: `Math.min(1, post.er / maxER)` where maxER is the highest ER in the dataset
         * Best-performing post (highest ER) gets the brightest appearance and closest position to planet center
         * Size: small (radius 0.02-0.05)
         * Color: `#E8D5A3` (cream-gold) for the brightest star, `#8892a0` (secondary text) for dimmer stars
       - Add subtle twinkle animation per UI-SPEC: opacity pulse between 0.6 and 1.0, random 3-8s cycle per star. Use `useFrame` with `Math.sin(clock.elapsedTime * speed + offset)`.

    4. Create `components/galaxy/GalaxyBackground.tsx` — deep void + ambient glow:
       - Dark void color: `#11131a` (handled by Canvas background)
       - Ambient glow: use a large, transparent sphere around everything with emissive `#374956` at very low intensity, or use drei's `<Environment>` with a dark preset
       - Keep it minimal — the background should feel infinite and dark
  </action>
